What is Grid?

Grid is 2D layout design in css. we can align items in column and row. mainly we can using for grid we can split the webpage in side by side.

Why we use grid? insteded of flex.

only can can align items in row or column in flex. so in grid we can align row and column sametime.

Grid properties

  • Display:grid - to activate grid layout
  • grid-template-columns-px,%,vh,vw,fr
  • grid-template-rows-px,%,vh,vw,fr
  • gap/row-gap/column gap:space between the items
  • grid-row/grid-column - rows and cloumn values.
